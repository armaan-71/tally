# Tally Frontend

This is the AI-native sales compensation dashboard for **Tally**, a proof-of-concept for the Driven.work internship application.

## Aesthetic: Claude/Notion Minimalist
- **Light Mode**: Clean, grayscale palette with high-contrast text.
- **Typography**: Inter (Sans-serif) for professional clarity.
- **Borders over Shadows**: Subtle dividers and border-based cards.

## Features
- **Payouts Dashboard**: Real-time commission stats and deal history.
- **Tally Agent**: Integrated AI assistant for querying financial data and policies.

## Tech Stack
- **Framework**: React 19 + Vite 6
- **Styling**: Tailwind CSS v4 (CSS-first architecture)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Ensure backend is running at `http://localhost:3000` (proxy configured in `vite.config.ts`).
